# yourworldoftext-ascii-paster

NodeJS script to paste any ascii art onto https://www.yourworldoftext.com/

# Usage

`node main.js`

art.txt is the file thats pasted

# Config

| Option      | Description    |
| ----------- | -------------- |
| World       | Target world   |
| x_offset    | X Tile offset  |
| y_offset    | Y Tile offset  |
